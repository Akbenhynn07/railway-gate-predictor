# Railway Gate Predictor

A machine learning system that predicts railway crossing gate closure times based on train speed, distance, and delay patterns.

The project demonstrates how predictive AI can be used to improve traffic flow and reduce waiting times at railway crossings.

## Features

• Train arrival simulation  
• Machine learning model training  
• Gate closure prediction system  

## Tech Stack

Python  
Pandas  
NumPy  
Scikit-learn  

## Project Goal

To explore predictive AI systems that can assist intelligent transportation infrastructure.
